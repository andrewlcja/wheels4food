Help Page

How do I make a request? You can make a request by following the steps below:

1. Click on Marketplace at the top of the page
2. Click on the checkbox of the items that you wish to request
3. Click Request for item(s) at the top of the page
4. Key in your desired quantity for each item in the Request Quantity column
5. Select Delivery Type
6. Select Delivery Date*
7. Click on Request

* Please note that delivery can only be made 3 days after today. How do I view the status of my request? You view the status of your request by following the steps below:

1. Click on My Inventory at the top of the page
2. You should see the status of each request under the Status column

Pending - Request has been submitted and is awaiting the Supplier's approval
Rejected - Request has been rejected, mouse over to see reason
Collection Created - Self Collection Request has been approved
Self Collection Completed - Self Collection has been completed
Self Collection Cancelled - Self Collection has been cancelled
Job Created - Job Request has been approved, awaiting volunteers to accept
Job Accepted - Job has been accepted by a volunteer
Job Completed - Job has been completed by a volunteer
Job Cancelled - Job has been cancelled
